# Brewfather

A homebrewing companion web app — recipe building, batch tracking, and fermentation logging for home brewers.

> ⚠️ Early-stage / in active brainstorming. Concept and scope are still being shaped.
> Note: "Brewfather" is a working title; an existing commercial app shares this name, so it may be renamed before launch.

## Status

🌱 Greenfield. Nothing built yet — currently defining the product concept, feature set, and tech stack.

## Ideas / Scope (WIP)

- Recipe builder (grain bill, hops schedule, yeast, water profile)
- Batch & brew-day tracking
- Fermentation logging (gravity, temperature, notes)
- Ingredient inventory
- Calculators (ABV, IBU, SRM, efficiency)

_This list is a starting point and will evolve as we brainstorm._
